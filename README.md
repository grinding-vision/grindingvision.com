# Grinding Vision Website

Official website for **Grinding Vision**, a company that develops mobile games.

## Features

- Company landing page
- Blog/news section powered by MDX
- SEO setup (sitemap, robots, JSON-LD)
- RSS feed
- Dynamic Open Graph images
- Tailwind CSS + Geist fonts

## Local Development

```bash
pnpm install
pnpm dev
```

## Production

```bash
pnpm build
pnpm start
```
